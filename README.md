Набор из пяти консольных математических игр:
- `brain-even` — проверка числа на чётность
- `brain-calc` — вычисление выражения
- `brain-gcd` — поиск наибольшего общего делителя
- `brain-progression` — поиск пропущенного числа в прогрессии
- `brain-prime` — проверка числа на простоту

## Минимальные требования

- Node.js `>= 20`
- npm `>= 10`
- Unix-like shell (Linux/macOS) или WSL

## Установка

```bash
make install
npm link
```

## Запуск игр

```bash
brain-games
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```

## Проверка качества кода

```bash
make lint
```
